Цепочки сравнения
 Операторы:
  and
  or
  not
   Пример and: оба условия должны быть True
   a = 3
   b = 4
   (a + b == 7) and (a + b != 8) нужно оборачивать в скобки
   print("hello world")
   Out:True

   Пример or: достаточно одного true
   a = 3
   b = 4
   (a+b == 7) or (a+b == 8)
   Out:True
   Пример not: меняет одно значение на противоположный
   12 == 12
   True
   not (12 == 12)
   False
